ФАН-ЗОНА

Сегодня 2 декабря 2014 года. Кассир начал распродажу билетов на концерт Металлики, но вот гавно... В фан-зону билеты так просто не поймать. Гребаные спекулянты и ажиотаж дают о себе знать

Решение проблемы

Я предлагаю вам супер софт для "ловли" билетов на крючок. Вот он:

function search(){ 
  $.ajax("https://msk.kassir.ru/kassir/event/view/12239").complete(function(resp){ 
    if(resp.responseText.indexOf("6500") != -1){
      window.clearInterval(window.id);
      alert("Фан-зона найдена!");
      window.location = "https://msk.kassir.ru/kassir/event/view/12239";
    }
  });
}

window.id = setInterval(search, 1000);

Как пользоваться

  1. Откройте страницу https://msk.kassir.ru/kassir/event/view/12239 в вашем любимом браузере
  2. Откройте консоль вашего браузера
  3. Вставьте код в консоль
  4. Далее не закрывайте вкладку, вылетит сообщение от том что есть свободные билеты. Пока ждете можете заниматься за компом чем угодно.
